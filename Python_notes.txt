recreate the built-in range() function with a class-based iterator
----------------------------------
class MyRange:

  def __init__(self, start, end):
    self.value = start
    self.end = end

  def __iter__(self):
    return self
    
  def __next__(self):
    if self.value >= self.end:
      raise StopIteration
    current = self.value
    self.value += 1
    return current
----------------------------------


generators            used to create iterators
                      they yield a value istead of returning a result
                      it keeps that state until the generator is run again and yields the next value
                      __iter__ and __next__ are created automatically
                      better performance, doesn't have all the values in memory

generator function    function that produces a sequence of results instead of a single value
                      simplifies the creation of iterators
yield()               causes the generator function to return a new value


generator function template
----------------------------------
def genfunc():
  for item in collection:
    yield epression
----------------------------------

generator function that recreates the MyRange class from above
----------------------------------
def my_range(start, end):
  current = start
  while current < end:
    yield current
    current += 1
----------------------------------

generator expression  like a list comprehension, but uses ()s instead of []s
                      returns a generator instead of a list
                      cannot be reused or restarted 
                      used for when you do not need a list/sequence

  listcomp = [x for x in range(11)]                         creates a list object
  genexpr = (x for x in range(11))                          creates a iterable generator object

sorting HOW TO
  list.sort()                 sorts a list in place, returns None

  sorted(list)                returns a newly sorted list
                              accepts any iterable
    key=                      a function that is used to determine the sort value of each item
    reverse=True              returns the sort in reverse order

examples 
  "".join(sorted('abcdabcdabcd'))               -->   'aaabbbcccddd'

def eval_expression(input_string):
    allowed_names = {"sum": sum}                              # dict of allowed names
    code = compile(input_string, "<string>", "eval")          # compile input string into bytecode
    for name in code.co_names:
        if name not in allowed_names:                         # check if name is allowed
            raise NameError(f"Use of {name} not allowed")     # raise NameError if not allowed
    return eval(code, {"__builtins__": {}}, allowed_names)

class Open_File():
  def __init__(self, filename, mode):
    self.filename = filename
    self.mode = mode
    
  def __enter__(self):                                     setup of the context manager
    self.file = open(self.filename, self.mode)
    return self.file
  
  def __exit__(self, exc_type, exc_val, traceback):       teardown of the context manager
    self.file.close()
    
with Open_File('filename.txt', 'w') as f:
  f.write('testing')

print(f.closed)

from contextlib import contextmanager

@contextmanager
def open_file(file, mode):
    try:
        f = open(file, mode)
        yield f
    finally:
        f.close()                                               teardown of the context manager
  
  
with open_file('filename.txt', 'w') as f:
  f.write("This is a sentence.")
  
print(f.closed)

special chars in regex    must be escaped in order to match them
  . ^ $ + ? { } [ ] \ | ( )
  
character classes
  \d                      any numeric digit from 0 to 9
  \D                      any char that is not a numeric digit
  \w                      any alpha, numeric digit or underscore char ("word" chars)
  \W                      any char that is not a "word" char
  \s                      any space, tab or newline char ("space" chars)
  \S                      any char that is not a "space" char
  
  \b                      word boundary
  \B                      not a word boundary, matches inside a word

  ^                       match occurs at the beginning of the string
    
  $                       match must occur at the end of the string
  
  r'^...$'                match the entire string
  
  []                      matches one char from this set
                          regex symbols do not need to be escaped

  [^ ]                    negative char class
                          matches all the chars NOT in the char class
  |                       pipe, will match any of the patterns (like OR statement)
                          Match object will be the first match

  ( )                     use backreferences to refer to search groups  
    .compile(r'(\w)\w*')  
    .sub(r'\1***', s_string)  

  quantifiers                          
  ?                       match 0 or 1 instances (optional group)
  *                       match 0 or more instances
  +                       match 1 or more
  {min, max}              match a group a min/max number of times
    (pattern){n}          match n instances
    (pattern){n,}         match n or more instances
    (pattern){,n}         match 0 to n instances
    (pattern){n,m}        match n to m instances

  greedy                  matches the longest string possible (Python regex default behavior)
  lazy (non-greedy)       matches the shortest string possible
    (pattern){3,5}?
  
  .                       wildcard char
                          any one char except a newline

  .*                      match anything (greedy, longest possible string)
  .*?                     match anything (non-greedy, shortest possible string)
